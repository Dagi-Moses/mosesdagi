Welcome to my personal portfolio!

## Live Demo

Check it out here: https://mosesdagi.vercel.app.com

---

## Features

- Responsive, mobile-first design
- Dark/Light theme toggle
- Showcase of personal, freelance, and company projects
- Smooth scroll navigation with active section highlighting
- Contact section with email and CV download

---

## Tech Stack

- **Frontend:** Next.js, React, TailwindCSS, TypeScript
- **Mobile:** Flutter (for showcased mobile projects)
- **Backend / APIs:** Node.js, Express, MongoDB (for some projects)
- **Deployment:** Vercel

---

## Getting Started

1. Clone the repo:

```bash
git clone <repo-url>
cd mosesdagi

2. Install Dependencies:

npm install
# or
yarn

3. Copy .env.example to .env.local

cp .env.example .env.local

4. Run the development server

npm run dev
# or
yarn dev


```
